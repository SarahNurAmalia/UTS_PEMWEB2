# Event Management Dashboard Invofest

## Deskripsi Project

Invofest Admin Dashboard merupakan website admin berbasis Fullstack yang digunakan untuk mengelola data event, category, dan speaker pada acara Invofest.  
Project ini dibuat menggunakan React Vite untuk frontend, Express.js untuk backend, Prisma ORM sebagai database toolkit, serta Supabase PostgreSQL sebagai database utama.

Admin dapat melakukan CRUD (Create, Read, Update, Delete) data secara realtime melalui dashboard yang telah terhubung dengan backend API.

---

# Features

- Menampilkan total category
- Menampilkan total speaker

## Speaker Management
